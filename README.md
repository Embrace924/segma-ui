# 欢迎使用 segma_ui

## Install
    npm i @segma/segma-ui

## Set
    2.2.0以上版本只抛出样式文件
    只需在引入element-ui的文件中引入改样式文件即可
    例如：
    element.js文件中
    import 'segma-ui/lib/segma-ui.css';//引入样式文件
    import {
        Pagination,
        Dialog,
        Autocomplete,
        Dropdown,
        ......
        }
