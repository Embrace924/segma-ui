# SegmaUI
## 简介
segma团队UI风格库

## 仓库地址
https://github.com/Embrace924/segma-ui.git

## 快速开始
### 1.安装
>2.3.7之后版本均发布在segma内部私有npm仓库内发布
 因安全原因，将不再npm上维护

```
npm i @segma/segma-ui
```
### 2.使用
>2.2.0以上版本只抛出样式文件
 只需在引入element-ui的文件中引入改样式文件即可
 例如：element.js文件中

```js
    import 'segma-ui/lib/segma-ui.css';//引入样式文件
    import {
        Pagination,
        Dialog,
        Autocomplete,
        Dropdown,
        ......
        }
```
